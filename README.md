# 🧠 Engagement Segmentation Explorer

A clustering-based tool for identifying participant engagement patterns and segments.

This project is part of the **Impact Analytics Suite**, a portfolio of decision-support tools focused on impact, programme, and data strategy workflows.

---

## 🎯 Overview

Different participants engage in different ways. This tool groups participants based on behavioural and outcome patterns to support targeted strategy.

---

## ✨ Key Features

- KMeans clustering  
- Segment profiling  
- Cluster visualisation  
- Behaviour-based grouping  
- Strategic interpretation  

---

## 🧠 Why I Built It

I built this to demonstrate how segmentation can uncover meaningful patterns in participant behaviour.

---

## 👥 Example Use Case

- Identifying highly engaged participants  
- Detecting at-risk groups  
- Designing targeted interventions  

---

## 🗂 Expected Data Structure

- participant_id  
- attendance_rate  
- completion_rate  
- satisfaction_score  
- engagement_frequency

## ⚙️ How to Run Locally

1. Clone the repository:
```bash
git clone https://github.com/TanikkaB/engagement-segmentation-explorer.git
```
2. Navigate into the project folder:
```bash
cd engagement-segmentation-explorer
```
3. Install dependencies:
```bash
pip install -r requirements.txt
```
4. Run the app:
```bash
streamlit run app.py
```
